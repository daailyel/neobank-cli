Oi! 👋  
Esse é um projeto simples que eu desenvolvi em Python pra praticar lógica de programação e entender melhor como funciona um sistema bancário.

💡 Sobre o projeto

O NeoBank CLI é um sistema de banco feito para rodar no terminal.

Ele simula coisas básicas de um banco, como criar conta, fazer login e movimentar dinheiro (depósito e saque).

A ideia foi treinar programação na prática e criar algo que parecesse um sistema real.

⚙️ O que ele faz

- Cria conta de usuário  
- Faz login com senha  
- Mostra saldo  
- Permite depósitos  
- Permite saques  
- Mostra histórico das transações  

🛠️ O que usei

- Python  
- JSON para salvar dados  
- Terminal (linha de comando)  
- Lógica de programação
